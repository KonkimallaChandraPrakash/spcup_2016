Intructions for running the MATLAB Application:
1.)Run the file 'trial1.m' in MATLAB from the submitted zip file
2.)In the GUI that appears , click on the 'Browse' button and browse for the wave file which you want to classify and then click on the 'Classify' button.
3.)A popup window will open up and it will display the classified grid and its confidence value.
4.)For 'None of these' case the confidence value will be zero.
